¿Qué es Git?
Git es un sistema de control de versiones distribuido diseñado para manejar proyectos de software de manera eficiente. Permite a los desarrolladores rastrear cambios en el código, colaborar en equipo y gestionar múltiples versiones de archivos de manera estructurada. Con Git, los usuarios pueden trabajar en diferentes ramas sin afectar el código principal, lo que facilita el desarrollo y la resolución de errores.
Características principales de Git
•	Distribuido: Cada desarrollador tiene una copia completa del historial del proyecto.
•	Eficiente: Maneja grandes proyectos de manera rápida y optimizada.
•	Seguro: Usa algoritmos criptográficos para garantizar la integridad de los datos.
•	Colaborativo: Facilita el trabajo en equipo mediante ramas y fusiones controladas.
Datos interesantes sobre Git
•	Git fue creado por Linus Torvalds en 2005 para gestionar el desarrollo del kernel de Linux.
•	Su nombre, "Git", es una expresión coloquial en inglés británico que significa "persona desagradable", en tono humorístico.
•	Es la base de plataformas populares como GitHub, GitLab y Bitbucket.
•	Empresas como Google, Microsoft y Facebook usan Git para gestionar su código fuente.
•	Git permite trabajar sin conexión, sincronizando cambios cuando sea necesario.
